Preparacion para el Examen de Permiso (DMV NY)

El examen escrito para una licencia clase D, M, DJ, MJ o E examina el conocimiento de las reglas viales, las técnicas de conducción segura, las señales viales y las leyes referentes al consumo de alcohol y drogas al conducir. Para aprobar el examen escrito, debe contestar correctamente al menos 14 de las 20 preguntas, pero debe contestar correctamente dos de las cuatro preguntas acerca de las señales viales.
Puede practicar para el examen escrito usando las preguntas incluidas en este Sitio de Web y el Manual para Conductores. Debe consultar constantemente el manual y no confiar únicamente en estas preguntas de práctica para estudiar.
